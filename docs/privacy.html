개인정보처리방침

시행일: 2026년 2월 5일

1. 수집하는 개인정보

수면 공식은 별도의 개인정보를 수집하지 않습니다. 모든 데이터(수면 기록, 카페인 섭취 기록)는 사용자의 기기 내 로컬 저장소에만 저장됩니다.

2. 데이터 저장 위치

  • 수면 기록: 기기 내 로컬 저장소 (localStorage)
  • 카페인 기록: 기기 내 로컬 저장소 (localStorage)
  • 설정 값: 기기 내 로컬 저장소 (localStorage)

3. 외부 전송

사용자의 데이터는 외부 서버로 전송되지 않습니다. 광고 서비스 제공을 위해 광고 네트워크(Google AdMob)가 사용될 수 있으며, 이는 토스 앱인토스 플랫폼의 정책을 따릅니다.

4. 데이터 삭제

사용자는 앱 내에서 직접 데이터를 삭제하거나, 브라우저의 로컬 저장소를 초기화하여 모든 데이터를 삭제할 수 있습니다.
